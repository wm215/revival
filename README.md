# Revival

Revival is a strength tracker built around Helms's Pyramid methodology.

## Overview
Revival helps you plan, log, and review training cycles with a structured, progression-first approach.

## Features
- Program structure aligned with Helms's Pyramid principles
- Workout/session logging
- Progress tracking over time
- Lightweight web UI

## Tech Stack
- JavaScript
- CSS
- HTML

## Quickstart
```bash
# install deps
npm install

# run locally
npm start
```

## Configuration
Create a local environment file if needed:

```bash
cp .env.example .env
```

Then set required environment variables for your local/dev setup.

## Usage
1. Start the app locally.
2. Create or load a training plan.
3. Log sessions and track progression metrics.
